Win10搭建Hexo个人博客

部署Hexo

  1. 安装node.js、git，配置git环境变量，可百度
  2. 创建一个目录用于存放博客相关文件
  3. 进入创建的目录，右键以管理员身份运行git bash,执行npm install hexo-cli -g，我输入npm install hexo -g会报错，无法安装，提示hexo不再维护
  4. 好像最新版的node.js已经自带npm，安装node.js的时候已经装了npm，但是为了保险还是执行一下 npm install，安装npm
    依次执行hexo init，hexo generate，hexo server
  5. hexo默认端口为4000，在浏览器中输入localhost:4000，查看效果，成功效果如下：


    部署到github

  6. 注册github账号，新建一个repository，名称为username.github.io，
    比如我的github用户名为godwhisper，则repository为godwhisper.github.io
  7. 复制repository的https地址，修改之前创建的目录下的_config.yml，修改deployment部分，如下所示：

  8. 设置SSH Keys，在git bash 输入ls -al ~/.sh，查看是否存在ssh keys，若存在，则删除.ssh文件里所有文件，不存在则没关系
  9. 输入如下指令（邮箱就是注册github时候的邮箱）后回车：

    ssh-keygen -t rsa -C “xxxxxx@sina.cn”

    然后提示输入passphrase(我没输入，如果输入的话，要记得，到时候要用)

    然后输入1.ssh-agent -s，2.ssh-add ~/.sh/id_rsa，回车后我这里出错了，若出错，则输入如下命令：

    eval ‘ssh-agent -s’

    ssh-add
  10. 接着添加SSH Keys到你的github账户，输入如下命令，拷贝C盘用户目录下的.ssh文件夹下的id_rsa.pub
  11. 进入github settings，点击SSH and GPG Keys，New SSH key，设置title，将刚才拷贝的内容粘贴到key中，点击Add SSH key即可
  12. 输入ssh -T git@github.com

    可能会有警告，不用管，输入yes就行

    完成部署

    依次键入命令：

    hexo generate

    hexo deploy

    OK!到此博客已搭建完成，浏览器中输入(当然是你的用户名):

    https://godwhisper.github.io/

    我的如下所示：



    每次修改本地文件后，都要在最初创建的目录下执行:

    hexo generate

    hexo deploy

    成功之后如下所示：

    至此，hexo博客搭建成功
